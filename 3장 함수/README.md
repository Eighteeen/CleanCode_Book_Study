## 🔍 리딩 도움
- _( p. 47 )_ `SRP(The Single Responsibility Principle)` : 하나의 클래스를 수정해야 한다면, 그 이유는 오직 한가지여야만 한다. (p.19 참조)
- _( p. 47 )_ `OCP(The Open Closed Principle)` : 클래스는 확장에 열려 있어야 하며 변경에 닫혀 있어야 한다. (p.19 참조)
